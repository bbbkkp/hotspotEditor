# hotspoteditor_Electron

## 安装
```
npm install
```

### 启动开发环境
```
npm run serve
```

### 运行Electron
```
elctron .
```

## 说明      

1. 选择全景文件夹的路径；
2. 全景主文件必须是tour.xml、tour.swf;
3. 测试版本不提供保存功能，仅保存修改内容到剪贴板，可根据需求进行操作。
4. 测试版本暂不提供添加热点功能。
5. 后续功能会逐渐完善（持续更新）。
